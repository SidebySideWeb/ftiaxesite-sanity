# ftiaxesite web

Astro 5 SSR frontend for [ftiaxesite.gr](https://ftiaxesite.gr), powered by Sanity CMS.

## Legal content

Privacy and cookie policy pages are rendered from Sanity `legalDoc` documents. **Have a qualified lawyer review all legal text before go-live.**

## Development

```bash
npm install
npm run dev
```

Requires `PUBLIC_SANITY_PROJECT_ID`, `PUBLIC_SANITY_DATASET`, and optionally `SANITY_WRITE_TOKEN` for quote form submissions.
